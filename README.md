Grupo 4: Gestión de servicios de Universidad / Cursos 

Este proyecto es un sistema de gestión para servicios universitarios. Su objetivo principal es administrar la información de estudiantes y los diferentes tipos de cursos que ofrece 
la institución. El programa funciona a través de una serie de clases, basada desde la clase base, este proyecto contiene la lógica para operaciones como inscripciones, actúa como la
interfaz principal que permite a los usuarios interactuar con el sistema.
